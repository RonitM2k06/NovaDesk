# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - Unreleased
### Added
- **Dashboard**: System information tracking (CPU, Memory, Disk).
- **Projects**: Asynchronous directory scanning and language detection.
- **Git**: Native Git operations (commit, stage, branch) without shell injection.
- **AI Workspace**: Streaming AI completion via OpenAI and Ollama.
- **Terminal**: Embedded SwiftTerm emulation with split panes and tabs.
- **Notes/Snippets**: SwiftData integration for local persistent storage.
- **Markdown**: HTML-bridged Live Preview with Mermaid and MathJax support.
- **Search**: Spotlight-style command palette and global inverted index.
