# Roadmap

This roadmap provides an overview of what the core maintainers are currently working on and what we plan to tackle next.

## Short-term (v1.1.0)
- Add Anthropic and Google Gemini to the AI Provider abstraction.
- Enhance Global Search with deep contents matching for Code Snippets.
- Add CI/CD GitHub Actions directly to the Workspace Automation UI.

## Medium-term (v1.5.0)
- Full Docker container management integration.
- Custom plugins and scripting extensions using JavaScriptCore.
- Cloud Sync for Notes and Snippets using CloudKit.

## Long-term (v2.0.0)
- iOS/iPadOS companion app for viewing notes and dashboards.
- Peer-to-peer real-time collaborative code editing.
