# Security Policy

## Supported Versions

Currently, only the `main` branch (v1.0.0+) is supported with security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please send an email to the repository owner. You should receive a response within 48 hours. If the vulnerability is accepted, we will work with you to patch it and publish a security advisory.

### Scope
- Malicious prompt injections via AI Workspace
- Unescaped shell execution in Git or Terminal Workspaces
- Sandbox escapes (if running in App Sandbox)
- Plaintext API key storage leaks
